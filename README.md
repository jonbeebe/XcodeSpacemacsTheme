# XcodeSpacemacsTheme
Spacemacs inspired theme for Xcode

## Installation
1. Download the zip or clone this repo
2. Open the folder using finder
3. Double tap **install.command** 
4. Restart your Xcode. 
5. Go to preference and see the scpacemacs_ theme.
6. Enjoy!
